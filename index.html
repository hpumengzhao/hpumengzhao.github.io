A. The Fair Nut and Elevator

题意：一栋n层的大楼，每层的人数为ai,每个人每天需要做两趟电梯，求电梯最初在哪一层才能节省电量。
solve：暴力枚举

B. Kvass and the Fair Nut

solve：二分一下就好了。

C. The Fair Nut and String

题意：给你一个字符串，让你求一个严格递增序列p1,p2,p3,…pk的个数，该严格递增序列满足：
1、a[p1…pk]=’a’.
2、对于每一个串中的i都有a[pi]<a[pos]<a[pi+1],a[pos]=’b’.
solve:把除’a’,’b’外的字母删掉，然后对这个字符串按照连续的’a’进行分块累乘再-1（因为有空串）。

D. The Fair Nut and the Best Path

题意：给你一棵树，每个顶点代表一个城市，你可以在第i个城市买w[i]的油，同时如果你想去下一个城市，那么你就会消耗掉
边权大小的油量，如果边权大于油量，你就不能走这条路，现在你要确定一条路径使得最终剩余的油量保持最多，输出最终最多的油量。
solve：我们令dp[y]代表以y为出发点时最大的剩余油量，那么如果当前油量大于等于边权的时候，x的父亲dp[y]=max(dp[x]+dp[y]-ww+w[x],dp[y]).DFS一下即可。

Educational Codeforces Round 55 (Rated for Div. 2)

发表于 2018-11-29 | 更新于 2019-01-11 | 分类于 codeforces

A. Vasya and Book（数学）

题意：一本书总共有n页，你现在读到了x页，你每次只能翻d页（前后都可以）,问你翻到y页至少需要多少次。
如果翻不到，则输出-1。
solve：分情况讨论即可。

B. Vova and Trophies(贪心)

题意：有一行字符串s，s只包含’G’和’S’,你最多只能交换一次两个字母的位置。求最长的连续’G’的个数。
solve:对字符串进行分块处理。记录每一段连续’G’的个数以及每两段的中间的’S’的个数。
对于字符串的处理只有两种：
1、给某一段的’G’的个数+1。
2、如果两段的间距为1，直接填过去。
这样遍历直接取最大即可。

C. Multi-Subject Competition（sort枚举）

题意：有n个人，m个题，你需要选出一些人去写这m个题，每个人能写的题目只有一道并且他们能得到的分数也是已知的。
要求就是选出来的人分配这m道题的时候，每道题分配的人相同（对于有的题目可以不分配人），求最大得分。、
solve:对于每个题目，先把写某一道题的人都存起来，然后对每种题的人进行从大到小排序，用一个数组记录对于
每道题选择相同人数的分数。取最大即可。

2018 Multi-University Training Contest 4（补题）

发表于 2018-11-27 | 更新于 2019-01-11 | 分类于 未分类

Problem B. Harvest of Apples

题意：计算c(n,0)+c(n,1)+c(n,2)+…c(n,m).
solve:令s(n,m)=c(n,0)+c(n,1)+c(n,2)+…c(n,m).则s(n,m)=2s(n-1,m)+c(n-1,m).
莫队离线查询即可。

[code lang=cpp]

#include<bits/stdc++.h>

using namespace std;
typedef long long ll;
const ll maxn = 1e5+100;
const ll mod = 1e9+7;
ll fact[maxn],inv[maxn],factinv[maxn];
ll anss[maxn];
struct query{
ll l,r,id;
}e[maxn];
void init(){
fact[0]=inv[1]=factinv[0]=inv[0]=fact[1]=factinv[1]=1;
for(int i=2;i<=maxn-10;i++){
fact[i]=(fact[i-1]%modi%mod)%mod;
inv[i]=((mod-mod/i)%modinv[mod%i]%mod)%mod;
factinv[i]=(factinv[i-1]%modinv[i]%mod)%mod;
}
}
ll c(ll n,ll m){
return (fact[n]%modfactinv[m]%modfactinv[n-m]%mod)%mod;
}
ll bl,ans;
ll n,m;
bool cmp1(query a,query b){
return (a.l/bl==b.l/bl)?a.r<b.r:a.l<b.l;
}
void add1(int pos){
ans=((2%modans%mod)%mod-c(pos-1,m)+mod)%mod;
}
void remove1(int pos){
ans=((ans%mod+c(pos-1,m))%mod*inv[2])%mod;
}
void add2(int pos){
ans=(ans%mod+c(n,pos))%mod;;
}
void remove2(int pos){
ans=(ans%mod-c(n,pos)+mod)%mod;
}
int main(){

ll t;
n=1;m=0;
ll L,R;
init();
//    cout<<inv[100000]<<endl;
scanf("%lld",&t);
bl=sqrt(t);
for(int i=1;i<=t;i++){
     scanf("%lld %lld",&e[i].l,&e[i].r);
     e[i].id=i;
}
sort(e+1,e+t+1,cmp1);
ans=1;
for(int i=1;i<=t;i++){
    L=e[i].l;R=e[i].r;
    while(n<L) add1(++n);
    while(n>L) remove1(n--);
    while(m<R) add2(++m);
    while(m>R) remove2(m--);
    anss[e[i].id]=ans;
}
for(int i=1;i<=t;i++){
    printf("%lld\n",anss[i]);
}
return 0;

}

B. Divide Candies（数学）

题意：有一个nxn的棋盘，行列都从1开始，对于坐标为(i,j)的格子，其中放入了糖果数量为i_i+j_j，现在有m个人
问你有几个格子的糖果数量S满足S%m==0；
solve：直接暴力肯定不行，但是我们观察到m的数据比较小，因此我们就从m开始切入。我们先O(m_m)的处理出
(x_x+y*y)%m==0,对于每一对满足条件的x,y我们都可以O(1)的求得在n内的i，j数量使得i%m==x,j%m==y.
将这些数量相乘累加即为答案。

D. Decorate Apple Tree

solve:我们用dp[i]代表i节点所包含的叶子节点数量，dfs处理之后排序输出即为答案（注意特判n==1的情况）。

6312: Game

题意：一个数组，里面n个数分别从1-n，每个人如果拿走一个数，那么这个数的所有因子都会被删除，直到有一方不能再拿数
为止，那么这个人就输了，对于输入的n，问先手是否能赢。
solve：10以内的写完发现先手一直赢，然后，直接输出”Yes”即可。

6315: Naive Operations

题意：给你两个数组a和b，a数组初始状态全部为0，对两个数组进行下列操作：
1、add l r 给a[l]-a[r]每个数+1。
2、query l r查询从l到r区间中[a[i]/b[i]]的和。
solve:线段树，我们令c[i]=[a[i]/b[i]]我们在线段树的每个节点上储存b数组的区间最小值以及c[i]的区间和。
初始状态的时候，c[i]全部为0，如果想要让c[i]+1，那么a[i]需要增加b[i]次，换个思路，即我们吧add操作改为让
b[i]-1，那么当该区间内的最小值变为0的时候，c[i]+1,我们在这个过程中需要用lazy来标记减去的次数，当b[i]变为0
的时候，再重新给b[i]赋值回去，lazy也变成0。

6318：Swaps and Inversions

题意：一个数组，每查询一组逆序对要花费x元，同时也可以花费y元交换连续两个数的位置，问查询完所有的逆序对至少花费的
金币数量。
solve：交换两个连续位置的数的本质就是消去逆序对，故最后的答案即为min(x,y) * 逆序对个数。
直接归并排序就好了。不过会卡cin，要关闭同步流。

2018 Multi-University Training Contest 1 补题（占坑）

6298：Maximum Multiple

题意：给你一个数n让你找到n的三个因子x,y,z使得x+y+z==n并且x_y_z最大
solve：只有n%3==0或者n%4==0时才可以，直接讨论即可。

6299:Balanced Sequence

题意：给你一堆括号字符串，让你把他们拼接起来，可以改变每个字符串的顺序，让你求出最大的匹配字符子字符串的长度：
solve：贪心，首先对于每一个字符串，去除掉那些已经可以配对的字符，然后剩下的字符我们按照每个字符串中’(‘和’)’的
贡献来进行排序：
1、对于字符串a和b，如果a的’(‘数量大于a的’)’的数量，b的’(‘数量小于b的’)’的数量，则a在b前面
2、如果a和b都是’(‘的数量大于’)’的数量，则谁的’)’数量多，谁在后面。
3、如果a和b都是’(‘的数量小于’)’的数量，则谁的’(‘数量多，谁在前面。
排完序后，我们直接模拟即可。

6300：Triangle Partition

题意：给你3n个点，让你构造n个不相交的三角形，每个点只能用一次，输出每个三角形所用点的编号。
solve：对所有点排个序，挨着的三个分配在一块就好了。

6301：Distinct Values

题意：让你构造一个长度为n的数组，使得这个数组满足输入的m对区间内的数不重复，并且字典序最小。
solve:set维护即可，set中的元素表示当前可用的元素，先对区间排序，对于区间[l,r],先从[1,l)获取可用元素，然后在进行更新，每用掉一个就erase掉，由于set自动排序，因此字典序是最小的。对于没有覆盖到的数组元素，全部赋值为1.

6308:Time Zone

题意：给你一个UTC+8时区的时间，转化成其他时区的时间。
solve:XJB讨论+模拟即可，代码有点丑。
